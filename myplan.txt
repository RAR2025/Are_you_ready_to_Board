**Tech stack additions I'd recommend:**
- **Drizzle ORM** over raw SQL — type-safe, works perfectly with Neon (serverless Postgres)
- **Zustand** for frontend state — far lighter than Redux for this use case
- **React Three Fiber** for the 3D code graph (Phase 3)
- **Unstructured.io** for parsing diverse doc formats in the ingestion pipeline
- **LangChain (Python)** for RAG chain orchestration in Agent 1
- **Neon** for serverless Postgres — free tier, branches per PR
- **Fly.io** for Python agents (Docker-native, cheap)
- **BullMQ + Redis** for async ingestion jobs so re-training doesn't block HTTP responses
